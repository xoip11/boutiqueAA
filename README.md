# BoutiqueAA 
Sitio web de moda con estructura modular:
- Navbar dinámico reutilizable
- Página principal
- Login independiente
- carpeta imagenes es donde van a ir todas las imagenes si quieres acceder a ella desde fuera de una carpeta en expecifico por ejemplo desde dentro del navbar es: "img src="imagen/
(nombre de la imagen)" alt="logo" width="70" height="70"" si es localmente si solo la ocupas mostrar ahi es: " img src="../BoutiqueAA/imagen/(nombre de la imagen)" alt="Vestido Largo"

## Estructura
- `/navbar` → contiene el menú
- `/login` → formulario de usuario
- `/css`, `/js` contiene los estilos y scripts por módulo
